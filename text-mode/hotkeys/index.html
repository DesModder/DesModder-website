Hotkeys

Accessibility

  • Escape, then Tab: focus the element after the editor ("Graph settings" menu)
  • Escape, then ShiftTab: focus the element before the editor ("Hide Expressions List" button)
  • Esc: exit menus

Tools

  • CtrlSpace: show completions (arrows to move, enter to accept)
  • Tab or Ctrl]: indent
  • ShiftTab or Ctrl[: dedent (un-indent)
  • CtrlF: search
  • CtrlShift[: fold selected structure
  • CtrlShift]: unfold
  • Ctrlclick: add cursor
